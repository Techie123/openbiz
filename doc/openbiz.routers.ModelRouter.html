<static> getDefaultRoutes(routePrefix, modelController, permission) → {object}

Generate default routes for standard data modal
Parameters:
Name Type Argument Description
routePrefix string the route name of this resource
modelController openbiz.controllers.ModelController the controller which map this route to
permission string <optional>
 the permission which default to protect this resource

Example
 //inside a module router e.g. /cubi/routes/account.js
 var routes = openbiz.ModelRouter.getDefaultRoutes('/accounts', openbiz.getController(cubi.account.AccountCountroller), 'cubi-account-manage');
 module.exports = routes;
